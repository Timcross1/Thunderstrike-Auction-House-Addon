Thunderstrike Market Agent Privacy Policy

Thunderstrike Market Agent uses a private backend service to answer questions about World of Warcraft auction-house and TradeSkillMaster market data.

Data Sent to the Service

When you use this GPT, your market-related question may be sent to the backend Market Agent so it can return auction-house summaries, item lookups, crafting profitability, posting opportunities, and related calculations.

Requests may include:

  • The text of your market question
  • Item names or profession names you ask about
  • The action endpoint being used
  • A private API credential used to authorize the request

Data Not Intentionally Collected

The service is not intended to collect personal information such as your real name, address, payment details, Battle.net login, or ChatGPT account details.

Users should not enter passwords, account credentials, or other sensitive personal information into this GPT.

Use of Data

The backend uses submitted requests only to provide market-analysis responses from the connected auction-house data.

The data is not sold to advertisers or shared with third parties for marketing.

Retention

Server logs may temporarily contain request metadata for debugging, security, and reliability. Logs are retained only as long as needed for debugging, security, and reliability.

Security

The backend requires an API key for access. Reasonable technical measures are used to limit unauthorized access, but no system can be guaranteed perfectly secure.

Contact

For privacy questions, contact the operator of this GPT.
